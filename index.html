Grameen Bank

provides microloans to women pursuing entrepreneurial endeavors currently living below the federal poverty line. Founded in Queens in 2008, this micro-finance-nonprofit organization is based on the Nobel Prize-winning of Muhammad Yunus' Grameen Bank.

The data on loan accounts from Grameen America was filtered to remove non-categorical columns. One-hot encoding of the remaining categories produced a sparse data set, and the dimensionality was reduced using principal component analysis. The resulting data set was grouped into five data sets using k-means clustering and compared to the original data for insight.
