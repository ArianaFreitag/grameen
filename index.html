Grameen Bank

is a global non-profit organization dedicated to helping entrepreneurial women who live in poverty build businesses to enable financial mobility. We are showing different groups of people that have received loans from the organization, in order to better understand how participants needs and backgrounds compare. Each point represents one enrolled individual.

?

The data on loan accounts from Grameen America was filtered to remove non-categorical columns. One-hot encoding of the remaining categories produced a sparse data set, and the dimensionality was reduced using principal component analysis. The resulting data set was grouped into five data sets using k-means clustering and compared to the original data for insight.
